# woowacourse 8기 최종 코딩테스트

## 기능구현 목록

### 편의점 전체 기능

- [ ] src/main/resources/products.md 를 읽어 상품 목록을 로드한다.
- [ ] src/main/resources/promotions.md 를 읽어 프로모션 목록을 로드한다.

- [ ] 인사와 함께 상품명, 가격, 재고, 프로모션 이름을 출력한다. 
    - 조건) 재고가 0개라면 "재고없음" 이라고 출력한다.
    - 조건) 금액은 천 단위로,를 붙여 출력한다.
- [ ] 구매할 상품명과 수량을 입력받는다. 
    - 예외) 형식에 맞지 않게 입력한 경우 ("[ERROR] 올바르지 않은 형식으로 입력했습니다. 다시 입력해 주세요.")
    - 예외) 재고에 존재하지 않는 상품을 주문한 경우 ("[ERROR] 존재하지 않는 상품입니다. 다시 입력해 주세요.")
    - 예외) 재고에 있는 상품(프로모션+일반)보다 많은 상품을 주문했을 경우 ("[ERROR] 재고 수량을 초과하려 구매할 수 없습니다. 다시 입력해 주세요.")
    - 예외) 기타 잘못된 입력의 경우 ("[ERROR] 잘못된 입력입니다. 다시 입력해 주세요.")
- [ ] 상품명과 수량을 통해 프로모션 상품, 일반 상품을 몇개씩 주면 되는지 계산한다. 
    - 조건) 오늘 날짜가 프로모션 기간 내에 포함된 경우에만 할인 적용한다.
    - 조건) 1+1 또는 2+1 프로모션이 각각 지정된 상품에 적용되며, 동일상품에 여러 프로모션이 적용되지 않는다.
    - 조건) 프로모션 기간중이라면, 프로모션 재고를 우선적으로 차감한다.
- [ ] 프로모션 적용이 가능한 상품에 대해 고객이 해당 수량만큼 가져오지 않았을 경우,
      더 받아갈지 여부를 물어보고, 입력받는다.
- [ ] 프로모션 재고가 부족하여 일부 수량을 프로모션 혜택 없이 결제해야 하는 경우
      일부 수량에 대해 정가로 결제할지 여부를 물어보고, 입력받는다.
- [ ] 위 두 입력으로 결과적으로 프로모션 상품, 일반 상품을 몇개씩 주면 되는지 다시 계산한다.
- [ ] 멤버십 할인 적용 여부를 입력받는다. 
- [ ] 영수증을 구성하고, 출력한다. (구매상품내역, 증정상품내역, 금액정보)
    - 조건) 멤버십 회원은 프로모션 미적용 금액의 30%를 할인받는다. 
    - 조건) 프로모션 적용 후 남은 금액에 대해 멤버십 할인을 적용한다.
    - 조건) 멤버십 할인의 최대 한도는 8000원이다.
- [ ] 재고에 일반상품, 프로모션 상품을 주문 수량에 맞게 차감한다.
- [ ] 추가 구매 여부를 입력받는다.
    - 조건) Y를 입력한다면 다시 인사와 함께 상품명, 가격, 재고, 프로모션 이름을 출력한다.
